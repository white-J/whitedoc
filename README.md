# 无序列表
无序列表用 - + * 任何一种都可以
- 列表内容1
+ 列表内容2
* 列表内容3
    - 无序嵌套

# 有序列表
注意要加空格
1. 列表内容
2. 列表内容
    1. 列表嵌套1
		2. 列表嵌套2
3. 列表内容
    - 无序嵌套1
    - 无序嵌套2
        - 无序3

# 表格
第二行分割表头和内容。
- 有一个就行，为了对齐，多加了几个
- 文字默认居左
- 两边加：表示文字居中
- 右边加：表示文字居右

注：原生的语法两边都要用|包起来。此处省略

示例：

表头|表头|表头
---|:--:|---:
内容|内容222222222|内容
内容3333333|内容|内容111111111111
